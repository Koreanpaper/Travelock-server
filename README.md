# Travelock-server

### 백엔드 팀원 : 김재근, 안태오, 주혜원, 한지윤

회원관련 -> 김재근, 한지윤<br>
기능관련 -> 주혜원, 안태오

사용기술
====
Java17<br>
SpringBoot3<br>
JPA<br>
QueryDSL<br>
MariaDB<br>
Redis<br>
Oauth2<br>
JWT<br>
Swagger<br>


Github 형상관리<br>
====
dev브랜치 fork후 개인 작업(모든 브랜치 가져오기해야 dev까지 fork됨)<br>
pull request 생성, pr 요청<br>
담당자(김재근) 검수 후 merge 또는 반려<br>
프로젝트 스테이지마다 회의 후 main브랜치에 병합
